Overview

The myFlix server-side component serves as a vital part of the myFlix web application, granting users access to a diverse range of movie details, including genres and directors. This project exemplifies full-stack JavaScript development, employing technologies such as Node.js, Express, MongoDB, and more. It encompasses a RESTful API, robust user authentication, and stringent data security.

Key Features

  • Comprehensive access to movie information, genres, and directors.
  • User-friendly registration and profile management.
  • Convenient addition and removal of movies from favorites.

Technical Details

For this project, I assumed the role of the lead developer and made decisions regarding the API's endpoints and their naming conventions. Looking back, I acknowledge that I could have benefited from a more detailed plan, especially in terms of endpoint functionality. This would have allowed me to avoid having to add additional endpoints later on to fulfill the intended functionality.

Potential Improvements

If I could revisit this project, I would pay closer attention to defining the functions I wanted to achieve in the end result. Specifically, I would be more meticulous in specifying the endpoints' naming and functionality to align with the desired functionality. This proactive approach would save time and prevent the need for additional endpoint modifications. Lessons Learned: Through this project, I gained a deep understanding of endpoints and their significance in creating a robust API. I learned the importance of specifying endpoint behavior accurately from the outset to streamline the development process and achieve the desired functionality.

Lessons Learned

This project represents a significant step in my journey as a developer, demonstrating my knowledge in full-stack JavaScript development and showcasing my ability to create a functional RESTful API with user authentication and data security.
