Overview

The myFlix Angular Client is the client-side component of the myFlix movie application. Developed using Angular, it complements the existing server-side codebase by providing a user-friendly interface for users to access movie information, directors, and genres. This project includes extensive documentation and project management using Agile techniques.

Role and Responsibilities

I served as the lead developer for this project, responsible for various tasks aimed at meeting the project's objectives. My responsibilities included setting up the Angular app, implementing user registration and login forms, designing the movie card component, defining routing, and providing comprehensive documentation using Typedoc and JSDoc. I also played a role in project management by maintaining the kanban board and incorporating feedback into the project.

Decisions Made and Outcomes

Throughout the project, I made several key decisions that influenced its development. One important decision was to use Angular Material for design, ensuring a consistent and visually appealing user interface. This choice aligned with best practices for creating responsive web applications.

Reflecting on the project, one area where I would do things differently is code consistency. Feedback highlighted instances where the same functionality was implemented in different ways within the same file, potentially leading to confusion and debugging challenges. In future projects, I would prioritize code consistency to enhance maintainability.

Lessons Learned

This project provided valuable insights into Angular development and the importance of clear documentation. I gained proficiency in Angular, including the use of Angular Material for UI design. Additionally, I honed my skills in project management using Agile techniques, such as maintaining a kanban board and estimating story points. One of the key takeaways from this project was the significance of code consistency. Maintaining a uniform coding style and approach within the project can significantly improve collaboration among team members and simplify debugging processes.
