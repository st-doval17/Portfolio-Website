Overview

The Pokedex App project involved creating a JavaScript web application that functions as a Pokedex. It loads data from an external API and allows users to view details of 150 different Pokemon. This project served as a practical demonstration of coding skills using HTML, CSS, and JavaScript.

My Role and Styling Decisions

In this project, I took on the role of the lead developer and made styling decisions to ensure the app's visual appeal. I aimed to create an aesthetically pleasing and user-friendly interface for displaying Pokemon data.

What I Would Do Differently

Looking back, one aspect I would improve is adding offline functionality. Ensuring that users can access some data even when offline would enhance the app's usability. This feature could be valuable for users in situations with limited internet connectivity.

Lessons Learned

This project provided valuable learning opportunities. I gained experience in integrating external APIs into web applications and using JavaScript to display data dynamically. Additionally, I learned how to enforce code quality using ESLint rules and successfully deployed the app to GitHub Pages, making it publicly accessible.
